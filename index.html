Featured

VDO.Ninja

Bring remote video into OBS via WebRTC

Peer-to-peer video streaming directly in your browser. No downloads, no accounts, no data collection. Used by NASA, AWS, and Twitch for professional remote production. Supports up to 30 guests per room with ultra-low latency.

Open VDO.Ninja Documentation GitHub

SocialStream.Ninja

Consolidate live chat from 100+ platforms

Aggregate chat from YouTube, Twitch, TikTok, Kick, Facebook, and more into a single dashboard. Features chat overlays for OBS, text-to-speech, AI chatbots, auto-responders, and cross-platform message relaying. Completely free.

About

Ninjafy Tools is a collection of software for video streaming and live production, created by Steve Seguin. What started as OBS.Ninja in 2018 has grown into a complete ecosystem used by over a million creators, broadcasters, and organizations worldwide.

Most tools are completely free with no accounts required. Many are open source with code available on GitHub. Some products like Meshcast.io offer optional paid tiers for heavy usage, but core functionality remains free.
